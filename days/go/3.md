# Day 3: Variables, Types, and Constants

## 📝 Today's Tasks

1.  **Declare Variables:** Practice `var`, short declaration `:=`, and type inference.
2.  **Basic Types:** Use integers, floats, booleans, and strings in small examples.
3.  **Constants:** Define typed and untyped constants; try `iota` for related constants.
4.  **Zero Values:** Print zero values for common types and explain when they matter.

## 📖 Task Explanations

Go has a small set of clear types and always initializes variables to a zero value. Short declaration (`:=`) is idiomatic inside functions; `var` is common at package scope. Constants and `iota` help express related values cleanly. Understanding zero values is essential for correctness—especially with pointers, slices, and maps.

## 💡 What You'll Gain

You will declare variables idiomatically, use constants and `iota`, and reason about zero values without surprises.

## 📚 Dictionary for Developers

*   **Zero Value:** The default value of a type when no explicit initialization is provided.
*   **Type Inference:** The compiler determining a variable's type from the right-hand side.
*   **`iota`:** An auto-incrementing constant generator used in const blocks.
*   **Short Declaration:** The `:=` syntax that declares and initializes a variable in one step.
