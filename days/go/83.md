# Day 83: TLS and Secure Transport

## 📝 Today's Tasks

1.  **HTTPS Locally:** Serve TLS with local certificates (mkcert or similar).
2.  **HTTP Client TLS:** Configure clients for TLS and timeouts.
3.  **HSTS Awareness:** Know what HSTS does at a high level.
4.  **Secrets:** Never commit private keys.

## 📖 Task Explanations

TLS protects data in transit. Local HTTPS catches mixed-content and cookie Secure-flag issues early.

## 💡 What You'll Gain

Ability to serve and call HTTPS endpoints with proper key hygiene.

## 📚 Dictionary for Developers

*   **TLS:** Transport Layer Security—encryption for network connections.
*   **Certificate:** A credential binding a public key to an identity.
*   **HSTS:** HTTP Strict Transport Security—forcing HTTPS in browsers.
