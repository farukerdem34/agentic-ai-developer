# Day 36: database/sql Basics

## 📝 Today's Tasks

1.  **Driver Setup:** Connect to PostgreSQL (or SQLite for local learning) via `database/sql`.
2.  **Ping and Lifecycle:** Ping on startup; close DB on shutdown.
3.  **Query/Exec:** Run simple queries and inserts.
4.  **Null Handling:** Use `sql.Null*` types where columns can be null.

## 📖 Task Explanations

`database/sql` is the standard database API. Drivers implement the interfaces. Always set connection pool limits and handle nulls explicitly.

## 💡 What You'll Gain

You will connect, query, and insert with `database/sql` safely.

## 📚 Dictionary for Developers

*   **`database/sql`:** Go's standard database API for SQL databases.
*   **Driver:** A package that implements database connectivity for a specific engine.
*   **Connection Pool:** A cache of database connections reused across requests.
