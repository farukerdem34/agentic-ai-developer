# Day 26: net/http Fundamentals

## 📝 Today's Tasks

1.  **Hello Handler:** Write an HTTP server with `http.HandleFunc` that returns plain text.
2.  **Request Inspection:** Log method, path, headers, and query parameters.
3.  **Status Codes:** Return appropriate status codes for success and client errors.
4.  **ListenAndServe:** Start the server and test with curl or a browser.

## 📖 Task Explanations

The standard library `net/http` is production-capable. Handlers are just functions; understanding the Request/ResponseWriter pair is enough to build real APIs.

## 💡 What You'll Gain

You will run a basic HTTP server and handle requests with clear status codes.

## 📚 Dictionary for Developers

*   **Handler:** A type or function that serves an HTTP request.
*   **`ResponseWriter`:** An interface for writing HTTP response headers and body.
*   **Status Code:** A numeric HTTP result such as 200, 404, or 500.
