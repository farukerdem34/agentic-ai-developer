# Day 90: Performance Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Profile your service, load test it, and land one meaningful optimization.
2.  **Report:** Write a short PERF.md with profiles, numbers, and changes.
3.  **Guardrails:** Add a benchmark or load script to prevent regressions.
4.  **Safety:** Ensure correctness tests still pass.

## 📖 Task Explanations

Performance without correctness is irrelevant. Documenting results makes the learning transferable.

## 💡 What You'll Gain

Evidence-based performance improvement with documentation and regression guards.

## 📚 Dictionary for Developers

*   **Regression Guard:** A test/benchmark that fails when performance worsens significantly.
*   **PERF.md:** A short report of performance investigation and results.
