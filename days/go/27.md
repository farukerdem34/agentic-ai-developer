# Day 27: Routing and Handler Patterns

## 📝 Today's Tasks

1.  **Manual Routing:** Route by method and path in a mux you control.
2.  **ServeMux:** Use `http.ServeMux` (Go 1.22+ enhanced patterns) or a small custom mux.
3.  **Handler Types:** Implement `http.Handler` on a struct with dependencies.
4.  **404/405:** Return proper not-found and method-not-allowed responses.

## 📖 Task Explanations

Clean routing and handler structs keep HTTP code maintainable. Prefer method-aware routes and inject dependencies via handler structs rather than globals.

## 💡 What You'll Gain

Structured routing and dependency-friendly handlers for small services.

## 📚 Dictionary for Developers

*   **ServeMux:** A request multiplexer that matches URLs to handlers.
*   **`http.Handler`:** An interface with `ServeHTTP(ResponseWriter, *Request)`.
*   **Dependency Injection:** Providing collaborators to a handler instead of hard-coding them.
