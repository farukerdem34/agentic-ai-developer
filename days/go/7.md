# Day 7: Methods and Receivers

## 📝 Today's Tasks

1.  **Value Receivers:** Add methods with value receivers and call them.
2.  **Pointer Receivers:** Add mutating methods with pointer receivers.
3.  **Consistency Rule:** Discuss when to choose value vs pointer receivers and keep them consistent.
4.  **Method Sets:** Observe how method sets differ for `T` and `*T`.

## 📖 Task Explanations

Methods attach behavior to types. Choosing value vs pointer receivers affects mutability and interface satisfaction. Idiomatic Go prefers consistency within a type's method set and uses pointer receivers when mutation or large structs are involved.

## 💡 What You'll Gain

You will attach methods correctly, mutate state safely with pointer receivers, and understand method sets.

## 📚 Dictionary for Developers

*   **Receiver:** The type parameter before a method name that binds the method to a type.
*   **Method Set:** The set of methods associated with a type, used for interface satisfaction.
*   **Mutation:** Changing the state of a value rather than returning a new one.
