# Day 97: Advanced Problem Solving

## 📝 Today's Tasks

1.  **Debug Challenge:** Diagnose a race, leak, or latency issue deliberately introduced.
2.  **Read Runtime Traces:** Use traces/profiles to justify your diagnosis.
3.  **Fix + Test:** Ship a fix with a regression test.
4.  **Writeup:** Explain root cause in a short postmortem format.

## 📖 Task Explanations

Expert engineers debug systematically. Evidence beats guesswork; postmortems capture learning.

## 💡 What You'll Gain

A disciplined debug cycle ending in a tested fix and a clear writeup.

## 📚 Dictionary for Developers

*   **Postmortem:** A blameless writeup of an incident's cause and fixes.
*   **Root Cause:** The underlying reason a failure occurred.
*   **Regression Test:** A test that fails if an old bug returns.
