# Day 13: Strings, Runes, and Bytes

## 📝 Today's Tasks

1.  **Bytes vs Runes:** Iterate a UTF-8 string by byte and by rune; compare results.
2.  **String Builder:** Build strings efficiently with `strings.Builder`.
3.  **Package strings/strconv:** Practice common conversions and helpers.
4.  **Immutability:** Explain why strings are immutable and when to use `[]byte`.

## 📖 Task Explanations

Go strings are immutable UTF-8 byte sequences. Correct Unicode handling uses runes. Efficient concatenation uses builders, not repeated `+` in loops.

## 💡 What You'll Gain

Correct Unicode-aware string handling and efficient string construction.

## 📚 Dictionary for Developers

*   **Rune:** A Unicode code point, represented as `int32`.
*   **UTF-8:** The variable-width encoding used by Go strings.
*   **`strings.Builder`:** An efficient way to build strings incrementally.
