# Day 20: Concurrency Basics Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Build a concurrent URL checker or worker that processes jobs with goroutines.
2.  **Channels + WaitGroup:** Coordinate work and collect results.
3.  **Cancellation:** Support cancel/timeout for the overall run.
4.  **Race-Free:** Pass `go test -race` (or run with `-race`).

## 📖 Task Explanations

Consolidate goroutines, channels, select, and sync into one small concurrent program. Correctness and clarity beat cleverness.

## 💡 What You'll Gain

Practical concurrent Go: fan-out/fan-in, cancellation, and race-free results.

## 📚 Dictionary for Developers

*   **Fan-out / Fan-in:** Spawning many workers then merging their results.
*   **Worker:** A goroutine that repeatedly processes jobs from a channel.
