# Day 40: Database Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Persist your REST API resources in a SQL database.
2.  **Repository + Tx:** Use repositories and at least one transactional flow.
3.  **Migrations:** Include migration files for the schema.
4.  **Tests:** Add integration tests with a test database or SQLite where practical.

## 📖 Task Explanations

Move off in-memory storage. Persistence, migrations, and tests demonstrate real backend skill.

## 💡 What You'll Gain

A CRUD API backed by SQL with migrations and solid data-access patterns.

## 📚 Dictionary for Developers

*   **Integration Test:** A test that exercises real collaborators like a database.
*   **Persistence:** Storing data so it survives process restarts.
