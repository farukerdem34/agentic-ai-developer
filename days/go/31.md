## 📖 Task Explanations

`context.Context` carries deadlines, cancellation, and request-scoped values. Passing ctx first is idiomatic. Never store contexts in structs long-term without care; prefer parameters.

## 💡 What You'll Gain

You will cancel work correctly and hook HTTP request cancellation into downstream calls.

## 📚 Dictionary for Developers

*   **Context:** A type that carries deadlines, cancellation signals, and request values.
*   **Deadline:** A time after which work should abort.
*   **Request-Scoped:** Data/lifetime tied to a single request.
