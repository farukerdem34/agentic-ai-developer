# Day 74: Kubernetes Basics for Go Services

## 📝 Today's Tasks

1.  **Deploy/Service Manifests:** Write a Deployment and Service for your API.
2.  **Probes:** Configure liveness/readiness probes to your health endpoints.
3.  **Resources:** Set CPU/memory requests and limits reasonably.
4.  **Config via Env:** Inject configuration with ConfigMap/Secret references.

## 📖 Task Explanations

Kubernetes runs containers at scale. Probes and resource settings are essential for reliable rollouts.

## 💡 What You'll Gain

Basic K8s manifests that run your Go API with probes and config.

## 📚 Dictionary for Developers

*   **Deployment:** A Kubernetes object managing replica Pods.
*   **Service:** A stable networking endpoint for Pods.
*   **Probe:** A Kubelet check for liveness/readiness/startup.
