# Day 15: File I/O and Encoding Practice

## 📝 Today's Tasks

1.  **Read/Write Files:** Read a file with `os`/`io` and write processed output.
2.  **Deferred Close:** Always close files with `defer` and handle close errors thoughtfully.
3.  **JSON Lines or CSV:** Encode/decode a simple structured format.
4.  **Practice Project:** Build a small file-processing tool that reports errors clearly.

## 📖 Task Explanations

Consolidate collections, errors, and I/O into a practical tool. Focus on resource cleanup, clear errors, and streaming where reasonable.

## 💡 What You'll Gain

Hands-on experience with files, encoding, and robust error reporting in a CLI tool.

## 📚 Dictionary for Developers

*   **`io.Reader` / `io.Writer`:** Core interfaces for streaming input and output.
*   **Resource Leak:** Failing to close files or other resources, exhausting system limits.
