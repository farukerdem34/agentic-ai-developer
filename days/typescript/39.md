# Day 39: Template Literal Types and String Manipulation Types

## 📝 Today's Tasks

1.  **Template Literals:** Create types like `on${Capitalize<EventName>}`.
2.  **Intrinsic String Types:** Use `Uppercase`, `Lowercase`, `Capitalize`, `Uncapitalize`.
3.  **Event Map Pattern:** Type an event emitter style API with string templates.
4.  **Route Patterns Lite:** Model simple route strings with templates.

## 📖 Task Explanations

Template literal types type dynamic strings. Ideal for event names, CSS-ish tokens, and routing patterns. Keep them bounded—unbounded string templates can explode the type checker.

## 💡 What You'll Gain

Practical template literal typing for events/routes without type-perf disasters.

## 📚 Dictionary for Developers

*   **Template Literal Type:** String types composed with `${}` patterns.
*   **Intrinsic String Manipulation Type:** Built-ins like `Uppercase<S>`.
*   **Type Explosion:** Combinatorial growth that slows the compiler.
