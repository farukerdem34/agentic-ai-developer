# Day 46: tsconfig Deep Dive

## 📝 Today's Tasks

1.  **target/module/moduleResolution:** Configure modern Node or bundler settings deliberately.
2.  **rootDir/outDir/include/exclude:** Control input/output cleanly.
3.  **strict Family:** Enable `strict` and explore subflags individually.
4.  **Write a Profile:** Create a recommended tsconfig for apps vs libraries.

## 📖 Task Explanations

tsconfig is architecture. Wrong `moduleResolution` wastes days. Maintain separate mental profiles for apps (bundler) and libraries (compatible emit).

## 💡 What You'll Gain

Confident tsconfig authorship for app and library scenarios.

## 📚 Dictionary for Developers

*   **moduleResolution:** Algorithm TS uses to resolve import paths.
*   **Bundler Resolution:** Resolution mode aligned with modern bundlers.
*   **Compiler Profile:** A known-good set of options for a project kind.
