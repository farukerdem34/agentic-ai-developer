# Day 83: Variance, Generics Safety, and Common Pitfalls

## 📝 Today's Tasks

1.  **Covariance Intuition:** Understand array readonly differences vs mutable arrays.
2.  **Function Parameter Contra-variance:** See why callback params are stricter.
3.  **Avoid Unsafe Generics:** Show a failing case with mutable arrays and subtypes.
4.  **Practical Rule:** Prefer `readonly` in public APIs to reduce variance footguns.

## 📖 Task Explanations

Variance explains many 'why won't this assign?' moments. Readonly arrays and careful callback typing prevent subtle bugs.

## 💡 What You'll Gain

Working intuition for variance and safer public generic APIs.

## 📚 Dictionary for Developers

*   **Variance:** How subtyping relates between generic containers.
*   **Covariance:** Containers that preserve subtype direction.
*   **Contravariance:** Containers/params that reverse subtype direction.
