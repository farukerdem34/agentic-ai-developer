# Day 37: infer and Type Extraction

## 📝 Today's Tasks

1.  **infer in Conditionals:** Extract inner types from wrappers (`T extends Promise<infer U> ? U : T`).
2.  **Extract Array Elements:** Infer element types from arrays/tuples.
3.  **Function Infer:** Infer return/params via conditional patterns (beyond utilities).
4.  **Build Unpack:** Create `Unwrap<T>` helpers for your containers.

## 📖 Task Explanations

`infer` declares type variables inside conditional arms—essential for unpacking nested structures. Prefer built-in utilities when they suffice; use `infer` for custom containers.

## 💡 What You'll Gain

Custom extraction helpers using `infer` for your own wrappers.

## 📚 Dictionary for Developers

*   **infer:** Keyword introducing a new type variable in a conditional extends clause.
*   **Unpack/Unwrap:** Extracting an inner type from a container type.
*   **Custom Container:** Your own generic wrapper type.
