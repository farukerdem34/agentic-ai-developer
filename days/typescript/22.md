# Day 22: Generic Constraints and keyof

## 📝 Today's Tasks

1.  **extends Constraints:** Restrict `T` with `T extends { id: string }`.
2.  **keyof Patterns:** Build `getProperty(obj, key)` safely typed with `K extends keyof T`.
3.  **Constrained Identity:** Practice constraints that still allow useful flexibility.
4.  **Error Reading:** Break constraints on purpose and interpret the errors.

## 📖 Task Explanations

Unbounded generics are too wide; constraints express requirements without losing reuse. `keyof` + generics enable type-safe property access—an everyday professional pattern.

## 💡 What You'll Gain

Constrained generics and type-safe property access helpers.

## 📚 Dictionary for Developers

*   **Generic Constraint:** `T extends Shape` limiting allowed type arguments.
*   **`keyof T`:** Union of keys of `T`.
*   **Indexed Access Type:** `T[K]`—the type of property `K` on `T`.
