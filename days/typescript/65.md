# Day 65: Async & Errors Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Build a typed async client (fake HTTP) with Result returns, timeouts, and domain errors.
2.  **No floating promises:** Lint/tests ensure callers handle results.
3.  **Table Tests:** Cover success, domain failure, timeout/abort.
4.  **Docs:** Explain error strategy in README.

## 📖 Task Explanations

Combine async + Result + abort into a realistic client. This pattern transfers to Nest/Next data layers.

## 💡 What You'll Gain

A typed async client with explicit errors, cancellation, and solid tests.

## 📚 Dictionary for Developers

*   **Floating Promise:** Unhandled Promise that lint should catch.
*   **Fake HTTP Client:** In-memory client simulating network calls.
*   **Error Strategy:** Documented approach for representing failures.
