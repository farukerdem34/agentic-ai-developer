# Day 26: Control Flow Narrowing Basics

## 📝 Today's Tasks

1.  **typeof Guards:** Narrow unions of primitives with `typeof`.
2.  **Truthy Narrowing:** Handle optional values with careful truthiness checks.
3.  **Equality Narrowing:** Use `===` checks against literals to narrow unions.
4.  **Early Returns:** Structure functions so narrowing persists via return guards.

## 📖 Task Explanations

Narrowing is how TypeScript tracks what a value must be after runtime checks. Prefer control-flow designs that make narrowing obvious: early returns beat deep nested ifs.

## 💡 What You'll Gain

Reliable primitive and equality narrowing without assertions.

## 📚 Dictionary for Developers

*   **Narrowing:** Refining a wide type to a more specific type via checks.
*   **Control Flow Analysis:** TS tracking types through if/return/switch paths.
*   **Truthy Check:** Using boolean coercion to exclude nullish/empty values (with care).
