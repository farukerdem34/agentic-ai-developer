# Day 74: Environment Config Parsing

## 📝 Today's Tasks

1.  **Schema for Env:** Parse `process.env` with Zod into a typed Config.
2.  **Fail Fast Boot:** Crash startup on invalid config.
3.  **Defaults & Coercion:** Apply defaults and numeric coercion safely.
4.  **No Live ProcessEnv Access:** Avoid scattering raw `process.env` reads.

## 📖 Task Explanations

Typed config at boot eliminates an entire production outage class. Centralize parsing; pass Config explicitly.

## 💡 What You'll Gain

Fail-fast typed environment configuration.

## 📚 Dictionary for Developers

*   **Fail Fast:** Aborting startup when config is invalid.
*   **Coercion:** Converting string env values into numbers/booleans.
*   **Config Object:** A typed, validated settings object.
