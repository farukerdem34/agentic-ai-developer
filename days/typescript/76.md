## 📖 Task Explanations

Architecture starts with import rules. Feature folders + thin public surfaces scale better than type-only layering debates.

## 💡 What You'll Gain

## 📚 Dictionary for Developers

*   **Barrel File:** An `index.ts` re-exporting a folder's public API.
*   **Feature Folder:** Code organized around a product feature.
*   **Dependency Direction:** Which modules are allowed to import which.
