# Day 4: Functions: Parameters, Returns, and Function Types

## 📝 Today's Tasks

1.  **Typed Functions:** Write functions with annotated parameters and return types.
2.  **Optional, Default, Rest:** Practice optional (`?`), default parameters, and typed rest args.
3.  **Function Type Expressions:** Store functions in variables typed as `(x: number) => string`.
4.  **Call Signatures in Objects:** Type an object that has a callable call signature or method.

## 📖 Task Explanations

Functions are TypeScript's first high-value surface: wrong arguments become compile errors. Optional parameters differ from `| undefined`; default parameters affect call-site requirements. First-class function types enable safe callbacks and dependency injection without classes.

## 💡 What You'll Gain

Confident typing of functions including optional/default/rest parameters and first-class function types.

## 📚 Dictionary for Developers

*   **Call Signature:** A type describing callable parameters and return type.
*   **Optional Parameter:** A parameter that callers may omit (`param?: T`).
*   **Rest Parameter:** Collects remaining arguments into a typed array.
*   **Callback:** A function passed as a value to be invoked later.
