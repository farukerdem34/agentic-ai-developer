# Day 36: Conditional Types

## 📝 Today's Tasks

1.  **Basic Conditional:** Write `T extends string ? A : B` examples.
2.  **Distributive Conditionals:** Observe distribution over unions.
3.  **Non-distributive Wrap:** Wrap with `[]` to prevent distribution when needed.
4.  **Practical Helper:** Implement a simplified `IsString<T>` style helper.

## 📖 Task Explanations

Conditional types enable type-level if/else. Distribution over unions is powerful and surprising—learn to control it. This is the gateway to advanced library typing.

## 💡 What You'll Gain

Working conditional types with intentional distributive behavior.

## 📚 Dictionary for Developers

*   **Conditional Type:** `T extends U ? X : Y` at the type level.
*   **Distributive Conditional:** Conditionals that map over union members automatically.
*   **Type-level Logic:** Computing types from other types.
