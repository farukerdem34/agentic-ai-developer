# Day 14: Index Signatures, Maps, and Key Precision

## 📝 Today's Tasks

1.  **Index Signatures:** Type open dictionaries with `[key: string]: V`.
2.  **Record Patterns:** Prefer `Record<'admin' | 'user', Permission>` when keys are known.
3.  **Mixing Known Props:** Combine explicit properties with index signatures carefully.
4.  **Keyof Prep:** List keys of an interface with `keyof` and map them in a helper.

## 📖 Task Explanations

Open index signatures are convenient and imprecise. Prefer finite key unions when possible. `keyof` unlocks safer dynamic property access and is a bridge to generics.

## 💡 What You'll Gain

Safer dictionary typing with precise keys instead of unchecked string indexes.

## 📚 Dictionary for Developers

*   **Index Signature:** Allows dynamic property names of a given key type.
*   **`Record<K, V>`:** Maps each key in `K` to value type `V`.
*   **`keyof T`:** A union of property names of `T`.
