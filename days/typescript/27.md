# Day 27: Type Predicates and Custom Guards

## 📝 Today's Tasks

1.  **User-defined Guards:** Write `function isString(x: unknown): x is string`.
2.  **Object Guards:** Guard unknown JSON into a typed interface with field checks.
3.  **Array Guards:** Build `isStringArray` style helpers.
4.  **Composition:** Combine guards for complex inputs.

## 📖 Task Explanations

Type predicates (`x is T`) teach the compiler what your runtime checks prove. They are essential at trust boundaries (I/O, JSON.parse, user input).

## 💡 What You'll Gain

Custom type guards that safely convert `unknown` into domain types.

## 📚 Dictionary for Developers

*   **Type Predicate:** A return type `x is T` enabling narrowing.
*   **Trust Boundary:** Where untrusted data enters your program.
*   **Runtime Proof:** Checks that justify a compile-time type refinement.
