# Day 38: Mapped Types

## 📝 Today's Tasks

1.  **Basic Mapped Types:** Build `{ [K in keyof T]: ... }` transforms.
2.  **Modifiers in Maps:** Add/remove `readonly` and `?` via mapping modifiers.
3.  **Key Remapping (as):** Rename keys with `as` clauses in mapped types.
4.  **Re-implement Partial:** Write your own `MyPartial<T>` and compare to the built-in.

## 📖 Task Explanations

Mapped types transform every property of a type systematically. They power most utility types. Key remapping enables elegant API redesigns at the type level.

## 💡 What You'll Gain

Mapped types including modifiers and key remapping.

## 📚 Dictionary for Developers

*   **Mapped Type:** A type that iterates keys to build a new object type.
*   **Mapping Modifier:** `readonly`/`?` added or removed while mapping.
*   **Key Remapping:** Using `as` to change keys during mapping.
