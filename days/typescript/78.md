# Day 78: Dependency Injection Lite without Frameworks

## 📝 Today's Tasks

1.  **Constructor Injection:** Pass dependencies explicitly.
2.  **Factory Composition Root:** Wire dependencies in a single `main`/composition file.
3.  **Avoid Service Locator:** Don't grab deps from global registries.
4.  **Testability Check:** Swap a dependency in tests easily.

## 📖 Task Explanations

You don't need a DI container. Explicit constructors + a composition root give most benefits with less magic—ideal TypeScript style.

## 💡 What You'll Gain

Manual DI with a clear composition root and easy test swaps.

## 📚 Dictionary for Developers

*   **Composition Root:** The place where the object graph is built.
*   **Constructor Injection:** Receiving dependencies via constructor params.
*   **Service Locator:** A global registry anti-pattern for retrieving deps.
