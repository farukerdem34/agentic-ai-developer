# Day 52: ESLint with typescript-eslint

## 📝 Today's Tasks

1.  **Setup:** Configure ESLint flat config (or legacy) with typescript-eslint.
2.  **Type-aware Rules:** Enable a small set of type-aware lint rules.
3.  **Ban any:** Add rules discouraging `any` and floating promises.
4.  **Lint Script:** Wire `npm run lint` and fix findings in your project.

## 📖 Task Explanations

typescript-eslint catches bugs types alone miss (floating promises, misuse of promises, unsafe anys). Keep rule sets pragmatic so developers trust lint.

## 💡 What You'll Gain

A working typescript-eslint setup with high-signal rules.

## 📚 Dictionary for Developers

*   **Type-aware Lint:** Lint rules using TypeScript type information.
*   **Floating Promise:** A Promise created but not awaited/handled.
*   **High-signal Rule:** A lint rule with few false positives.
