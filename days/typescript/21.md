# Day 21: Generic Functions

## 📝 Today's Tasks

1.  **Identity Generic:** Implement `<T>(value: T) => T` and use with multiple types.
2.  **Generic Arrays Helpers:** Type `first`, `last`, or `unique` helpers with generics.
3.  **Infer from Usage:** Call generics without explicit type args and observe inference.
4.  **Explicit Type Args:** Pass `<Type>` explicitly when inference cannot decide.

## 📖 Task Explanations

Generics parameterize types the way functions parameterize values. They preserve relationships between inputs and outputs—the core of reusable typed utilities.

## 💡 What You'll Gain

Write and call generic functions with solid inference habits.

## 📚 Dictionary for Developers

*   **Generic Type Parameter:** A placeholder type like `<T>` filled in by callers/inference.
*   **Type Argument:** The concrete type provided for a generic parameter.
*   **Inference from Usage:** TS figuring out `T` from the values you pass.
