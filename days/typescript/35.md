# Day 35: Utility Types Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Build layered types for an entity: DbUser, PublicUser, UserUpdate, UserCreate.
2.  **Derive Don't Duplicate:** Prefer Pick/Omit/Partial/Readonly over copy-paste shapes.
3.  **Map Configs:** Use Record for typed feature flag or permission tables.
4.  **Document Coupling:** Note which derived types intentionally track a source type.

## 📖 Task Explanations

Professional codebases maintain multiple shapes per entity. Utility types keep them honest if you document the relationships.

## 💡 What You'll Gain

A layered type system for one domain entity using utility types end to end.

## 📚 Dictionary for Developers

*   **Layered Types:** Different type shapes per architectural layer.
*   **Derived Type:** A type computed from another type.
*   **Source of Truth Type:** The base type others derive from.
