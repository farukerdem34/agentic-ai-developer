# Day 5: TypeScript Fundamentals Practice Project

## 📝 Today's Tasks

1.  **Project Goal:** Build a small typed utility module (e.g., grade calculator, password strength helpers, or unit converters).
2.  **Strict Public API:** Export functions with explicit parameter and return types; forbid `any`.
3.  **npm Scripts:** Add `build` (`tsc`) and `start` scripts; ensure a clean compile.
4.  **README:** Document how to install, build, and run; list the Types you exported.

## 📖 Task Explanations

Consolidation day: combine toolchain, annotations, primitives, and functions into a complete micro-package. Treat exported types as a contract—future you (and teammates) will depend on them.

## 💡 What You'll Gain

A strict-mode mini-project with a typed public API and a repeatable build workflow.

## 📚 Dictionary for Developers

*   **Public API:** Exported functions/types consumers are expected to use.
*   **Strict Mode:** `strict: true` enabling safer compiler defaults.
*   **Utility Module:** A focused package of pure helpers without UI/framework coupling.
