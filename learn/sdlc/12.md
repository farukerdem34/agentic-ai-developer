### CI Purpose

CI triggers automated tests instantly upon a code commit. Automated quality gates remove human error and bias by holding all developers to the exact same thresholds. CI pipelines embed DevSecOps practices by running automated gates like SAST, DAST, and vulnerability assessments before deployment.

### Rollback Plan

| 3 | Users see standard LLM responses with message: "Tool calling is under maintenance" |
| 4 | Investigate root cause in staging |
| 5 | Fix, test, re-deploy |
